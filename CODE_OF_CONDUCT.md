# Code of Conduct

This project follows the Contributor Covenant Code of Conduct.

## Our Standards

- Be respectful and professional.
- Assume positive intent and focus on technical outcomes.
- Do not harass, insult, or discriminate.

## Enforcement

Project maintainers are responsible for clarifying and enforcing this policy and may remove, edit, or reject comments, commits, code, wiki edits, issues, and other contributions that are not aligned with this Code of Conduct.
